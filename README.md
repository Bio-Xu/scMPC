# scMPC
scMPC, a transfer-learning framework that leverages metastasis-associated hallmarks with matched primary and metastatic bulk transcriptomes to systematically identify metastatic potential cells (MPCs) at single-cell resolution.
Briefly, the scMPC framework began by performing systematic feature selection and optimization on bulk transcriptomic datasets, establishing a robust panel of 19 metastasis-discriminative gene signatures (M_signatures). These signatures were then incorporated as guiding features for single-cell analysis, enabling the recognition of transcriptional programs associated with metastatic potential.
